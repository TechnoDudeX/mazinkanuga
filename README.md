# mazinkanuga.github.io

Personal resume and portfolio site for Mazin Kanuga — Lifecycle Marketing Manager in crypto and fintech.

## About

A single-page resume site built with vanilla HTML and CSS. No frameworks, no dependencies — just a fast, self-contained file with everything inline.

## What's on the page

- **About** — Short bio and career stats
- **Experience** — Kraken, Solidus Labs, Coinsquare
- **Projects** — Things built and shipped
- **Skills** — Tools and disciplines
- **Beyond Work** — Maz's Burgers and Ramadan Sammich

## Stack

- HTML / CSS / Vanilla JS
- Fully self-contained (images base64 encoded inline)
- No build step, no dependencies
